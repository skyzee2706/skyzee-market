# Prerequisites

Before you can run SKY Market locally, set up the following:

## Node.js

Install **Node.js v18 or higher**:
- Download: [nodejs.org](https://nodejs.org)
- Verify: `node --version`

## MetaMask Wallet

1. Install [MetaMask](https://metamask.io) browser extension
2. Add **Sepolia Testnet** to your networks
3. Get free test ETH: [sepoliafaucet.com](https://sepoliafaucet.com)

## Sepolia RPC Endpoint

Get a free RPC URL from one of:
- [Infura](https://infura.io)
- [Alchemy](https://alchemy.com)

## WalletConnect Project ID

Register a free project at [WalletConnect Cloud](https://cloud.walletconnect.com) and copy your Project ID.
